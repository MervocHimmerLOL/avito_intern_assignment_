# Инструкция по запуску автотестов
# Шаг 1. Установите репозиторий командой .
```
git clone https://github.com/MervocHimmerLOL/avito_intern_assignment_
```
# Шаг 2. Убедитесь, что на вашем компьютере установлен Python. В командной строке выполните команду.
```
python -v  
```
# Шаг 3. Перейдите в корневую папку проекта
В командной строке выполните:
```
cd avito_intern_assignment_
```
# Шаг 4. Установите необходимые зависимости из файла requirements.txt
Для этого выполните команду 
```
pip install -r requirements.txt
```
# Шаг 5. Запустите автотесты.
Находясь в корневой папке(avito_intern_assignment), выполните команду 
```
pytest task_2/ -v
```
Либо, если находитесь в папке task_2, выполните команду 
```
pytest -v
```
